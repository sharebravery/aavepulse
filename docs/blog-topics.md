# AavePulse 博客选题

> 说明：这里记录可公开表达的主题方向，博客正文本身不提交到 Git。

## 适合写成博客的主题

1. 为什么选择 Aave 作为 Web3 Go 演示项目
2. 一个后台模板如何被改造成一个 DeFi 后台
3. The Graph 在链上数据索引里的作用
4. 只读 DeFi 后台的架构怎么设计
5. React 仪表盘如何展示链上快照和告警
6. 为什么这个项目不做交易签名和私钥处理

## 每篇文章建议包含的要点

- 背景：为什么要做这件事
- 方案：考虑过哪些选择
- 取舍：为什么最后选这个方向
- 结果：项目最终呈现什么能力
- 经验：做这个项目时学到了什么
